# calendar
根据作息表生成日历，能在手机上提醒自己日程的项目
## 前言（ldy的碎碎念）
众所周知，在googler日历上只能一项一项地添加你所需要的任务、课程，作为一名学生我经常会不停地在日历和课程表里一直跑来跑去，这很麻烦，所以我打算把我的课程表搞进我的日历里，然后有什么特别的事情可以再导入进去。因为我的课表是双周循环的，所以我可以输入双周后循环生成之后每天的日程。特别的是，google日历还可以在半小时前提醒用户，这就使得我们的使用体验很棒。
基本上所有市面上的日历都支持导入ics格式的日历，这边主要讲的是google日历，其他用户请自行查找导入的入口。

## 版本
v 1.0 因为我发现如果要从外界导入文件的话需要严格的格式要求，这显然不利于用户操作，所以我现在的想法是用户直接在代码里编写自己的日程并做添加，这样方便修改。
